To use the sequencer :

  • Right or shift click to create a starting node
  • Drag right click to redirect the signal
  • Wheel or control click to toggle the keys
  • Click on the start button

This is an hexagonal isomorphic keyboard sequencer inspired by electrogear and Musix Pro.

Depending on the layout selected, each of the three axes corresponds to a different interval.

Each complete layout can be encoded into an unique UIL label, you can learn more in this paper.
